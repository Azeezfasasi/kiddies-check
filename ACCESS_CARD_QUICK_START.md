# Student Access Card - Quick Start Guide

## 🎯 What is the Access Card?

The Access Card is a professional student ID card that includes:
- Student photo placeholder with icon
- Student name and information
- Student ID number
- Date of birth
- Grade/Class level
- Email address
- **QR Code** - for teachers to scan and mark attendance
- School name

## 🚀 How to Use

### Step 1: Navigate to Students Management
1. Log in to your dashboard
2. Go to **Students Management** or **All Students** page
3. You'll see a list of all students in your class/school

### Step 2: View a Student's Access Card
**Desktop Version:**
1. Find the student in the table
2. Look for the **Card icon** (looks like a card symbol) in the Actions column
3. Click it to open the card modal

**Mobile Version:**
1. Find the student in the cards list
2. Click the **"Card" button** in the action buttons section
3. The access card modal will open

### Step 3: Download the Card
1. A modal popup will appear showing the beautifully formatted access card
2. Review the card to ensure all information is correct
3. Click the **"Download Card"** button
4. The card will be saved as a PNG image file

### Step 4: Print and Distribute
1. Open the downloaded image file
2. Print the image on cardstock or regular paper
3. Cut to desired size (standard ID card: 3.375" x 2.125")
4. Give the card to the student
5. Laminate for durability (optional)

## 📋 What Information is on the Card?

| Field | Source |
|-------|--------|
| Student Name | From student profile |
| ID No | Student Number (studentNo) |
| DOB | Date of Birth |
| Grade | Grade Level or Class |
| Email | Student email address |
| QR Code | Encoded with student ID & school info |
| School Name | Your school name |

## 📱 Using the QR Code

### For Teachers (Attendance Marking):
1. Get a QR code scanner app on your smartphone or tablet
2. When a student arrives, scan their access card's QR code
3. The QR code contains:
   - Student ID
   - Student name
   - School ID
   - Timestamp when scanned

### For Future Integration:
The QR code can be integrated with an attendance system to:
- Automatically mark attendance
- Track arrival/departure times
- Generate attendance reports
- Send notifications to parents

- **Quality:** High resolution (suitable for printing)

## ⚠️ Troubleshooting

### "Card icon is missing on desktop"
- Make sure you're logged in as a teacher/admin
- Refresh the page
- Check browser compatibility

### "Download button doesn't work"
- Check if pop-ups are blocked in your browser
- Try a different browser
- Try the print option instead

### "QR code won't scan"
- Ensure QR code is fully visible in the printed card
- Try printing with better quality settings
- Use a different QR scanner app
- Check that the printed size isn't too small

### "Student information shows N/A"
- Ensure the student record has all fields filled (Name, ID, DOB, etc.)
- Go back and edit the student profile to add missing information
- Refresh and try again

## 🔐 Data Privacy

- QR code contains only non-sensitive identifiers (student ID, school ID)
- No passwords or tokens are encoded
- Download happens on your device only (not sent to servers)
- Card can be printed and physically secured

## 💡 Tips & Best Practices

1. **Print on Quality Paper:**
   - Use cardstock (65-110 lb recommended)
   - For durability, laminate the cards
   - Print in color for better appearance

2. **Card Size:**
   - Standard ID card: 3.375" x 2.125"
   - Can be printed larger for visibility
   - Smaller cards for wallets

3. **Bulk Printing:**
   - Download one card at a time
   - Consider bulk printing system for future
   - Keep backup digital copies

4. **Attendance Tracking:**
   - Keep QR scanner accessible
   - Train staff on scanning process
   - Regular backup of attendance data

5. **Card Replacement:**
   - If card is lost/damaged, easily generate a new one
   - No additional cost to generate new cards
   - Same process as first generation
